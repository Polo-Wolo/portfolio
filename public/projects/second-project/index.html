Control Flow

if statements

  • Form 1: if
if name == 'Homer':
    print('Hi, Homer.')
  • Form 2: if/else
if name == 'Homer':
    print('Hi, Homer.')
else:
    print('You are not Homer')
  • Form 3: if/elif/else
if name == 'Homer':
    print('Hi, Homer.')
elif name == 'Marge':
    print('Hi Marge!')
else:
    print('You are not Homer or Marge')

for Loops

The for loop iterates over a list, tuple, dictionary, set or string.

for indice in range(5):
    print('Hi, Homer.')

while Loops

spam = 0
while spam < 5:
    print('Hi, Homer.')
    spam = spam + 1

Functions / Procedures

Functions and Procedures are specified using the def keyword.

def say_hello(name):
    message = "Hi, {} !".format(name)
    print(message)

say_hello("Homer")

Optional Argument

def say_hello(name="Homer"):
    message = "Hi, {} !".format(name)
    print(message)

say_hello()  # no argument: default value if used for name
say_hello("Marge")

Return Values

def write_message(name):
    message = "Hi, {} !".format(name)
    return message

message = write_message("Homer")
print(message)
